# Greenden

Greenden is a responsive plant store website built using HTML and Tailwind CSS. The project features a clean and modern design with three main pages: Home, Products, and Contact Us. The website is fully responsive and provides a smooth user experience across desktop, tablet, and mobile devices.

## Features
- Responsive design for all screen sizes
- Modern and user-friendly interface
- Home page with featured content
- Products page showcasing plant collections
- Contact Us page for customer inquiries
- Built using Tailwind CSS utility classes
- Clean and organized code structure

## Technologies Used
- HTML5
- Tailwind CSS

## Pages
- Home
- Products
- Contact Us

## How to View
1. Clone the repository:
   ```bash
   https://jayashree-s1204.github.io/Greenden-tailwind/
   ```
2. Open the project folder.
3. Launch `index.html` in your web browser.
