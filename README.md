# Greenden

Greenden is a responsive plant store website built using HTML and Tailwind CSS. The project features a modern and clean user interface designed to provide an engaging user experience across all devices. The website consists of three main pages: Home, Products, and Contact Us, allowing users to explore plant collections and get in touch easily.

## Live Demo

View the project here:

https://jayashree-s1204.github.io/Greenden-tailwind/

## Features

- Fully responsive design
- Modern UI built with Tailwind CSS
- Home page with featured sections
- Products page showcasing plant collections
- Contact Us page with contact form
- Mobile-friendly navigation
- Clean and organized layout

## Technologies Used

- HTML5
- Tailwind CSS


2. Open the project folder.

3. Run `index.html` in your browser.
